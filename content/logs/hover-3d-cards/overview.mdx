# Hover-Reactive 3D Cards

A responsive UI demo showcasing interactive 3D card hover effects using CSS perspective and motion spring animations. Built with Next.js, TailwindCSS, and Framer Motion.

## Project Overview

This project serves as a foundation for exploring real-time 3D experiences, focusing on micro-interactions and motion principles. The goal is to master these concepts before moving on to more complex 3D tools like Spline and Three.js.

## Technical Stack

- **Framework**: Next.js 14
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Content Management**: ContentLayer (MDX)
- **Type Safety**: TypeScript

## Implementation Details

### Core Components

1. **Card Component**
   - Uses CSS perspective for 3D effect
   - Implements hover detection with Framer Motion
   - Applies spring animations for smooth transitions
   - Features depth-based shadow effects

2. **Card Grid**
   - Responsive grid layout using TailwindCSS
   - Dynamic card sizing based on viewport
   - Smooth reflow animations

### Animation System

The project implements a layered animation approach:

```tsx
// Example of card hover animation
const cardVariants = {
  hover: {
    rotateX: 10,
    rotateY: 10,
    scale: 1.05,
    transition: {
      type: "spring",
      stiffness: 300,
      damping: 20
    }
  }
};
```

### Performance Considerations

- Used `transform` properties for optimal performance
- Implemented `will-change` hints for smoother animations
- Optimized re-renders with proper React hooks usage
